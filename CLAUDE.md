## AI Law Firm - משרד עורכי דין חכם

משרד עורכי דין ישראלי מבוסס AI לייצור מסמכים משפטיים, ניהול תיקים ומחקר משפטי.

## שפת עבודה

- **שפה:** עברית
- **כיוון טקסט:** RTL (ימין לשמאל)
- כל המסמכים, התבניות והפלט נכתבים בעברית אלא אם צוין אחרת

## מבנה הפרויקט

```
.claude/agents/      # סוכני AI לביצוע משימות משפטיות
.claude/commands/    # פקודות מותאמות אישית
.claude/skills/      # מיומנויות (skills) של Claude
templates/           # תבניות מסמכים לפי תחום
  general/           # ייפוי כוח, הסכם שכ"ט, מכתב התראה
cases/               # תיקים פעילים
clients/             # לקוחות
research/            # מחקר משפטי
```
